The attack

Indirect Prompt Injection

An attacker embeds hidden instructions inside a document, API response, or web page that an AI agent will read. The agent's tool call is legitimate. The network request is valid. Only the content is malicious — and it arrives from a trusted tool, in the direction nobody inspects.

The injected instruction silently re-programs a legitimate, authorized agent mid-session. No login. No exploit. Just a poisoned document.

What existing tools see

✅ Valid HTTPS connection
✅ Authorized agent
✅ Permitted tool call
✅ Normal-looking response
✗ Injection inside the content

Why it works

Perimeter firewalls inspect addresses and protocols — not meaning. SAST scans code — not runtime behavior. Model-level guardrails read the user message — not the tool response.

Fast path — enforces

Deterministic inspectors run inline on every call and response. First non-allow verdict wins. Any inspector failure → block. Never silent pass-through.

Parallel path — advises

LLM sentinels analyze semantics, context, behavior across the session. They can only signal the circuit breaker — they can never directly enforce. An LLM cannot override a deterministic decision.

The core principle

The law that governs everything

Agents provide intelligence.
Deterministic systems enforce authority.

Why this matters

An LLM can be prompt-injected into approving a bad verdict. It can be confused, hallucinated, or manipulated. A deterministic circuit breaker cannot. The hard security decisions — block, quarantine, Siege — are never touched by an LLM.

What it means in practice

LLM sentinels produce evidence. The deterministic circuit breaker decides what to do with it. A compromised sentinel can at worst produce a bad signal — it cannot bypass enforcement. The wall holds even if the intelligence is wrong.

Runtime response

Three operating modes.
The system responds to attacks as they escalate.

NORMAL

Standard monitoring. Normal containment thresholds. Scheduled red-team drills. Minimal overhead. Clean traffic flows freely.

SUSPICIOUS

Containment threshold halved — sessions quarantined sooner. High-risk calls held for operator approval. Sentinels elevated. Triggered automatically when a session is quarantined.

SIEGE

Threshold collapses to one block. Live JWT tokens revoked for all quarantined sessions — a contained agent cannot re-authenticate to escape. Commander bulk-freezes credentials. Only a capability-gated human de-escalates.

Mode escalates monotonically and automatically. De-escalation requires a human with an olive:command token. The Security Commander is deterministic code — not an LLM.
